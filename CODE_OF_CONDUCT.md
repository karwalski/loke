# Code of Conduct

## Our Commitment

The loke project is committed to providing a welcoming, respectful, and inclusive environment for everyone who participates — regardless of experience level, background, identity, or perspective.

We adopt the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as our code of conduct. The full text is available at that link and applies to all project spaces, including the GitHub repository, issue tracker, pull requests, discussions, and any other communication channels.

## Summary

We expect all participants to:

- Be respectful and considerate in all interactions
- Welcome newcomers and help them get started
- Accept constructive feedback graciously
- Focus on what is best for the community and the project
- Show empathy towards other community members

## Scope

This code of conduct applies to all project spaces and to any individual officially representing the project in public spaces.

## Enforcement

Instances of unacceptable behaviour may be reported to the project maintainers at **conduct@tokelang.dev**. All reports will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of the reporter.
